# Simple-Landing-page
---
A beginner-friendly landing page built using HTML and CSS.
It includes a basic navbar and a hero section, designed based on a layout created in Figma.
This project focuses on practicing structure, layout, and styling fundamentals.

⭐ Features<br/>
▪️Simple and clean layout

▪️Basic navigation bar

▪️Hero section with heading, subheading, and button

▪️Design inspired from Figma

▪️Beginner-friendly HTML & CSS codebase
